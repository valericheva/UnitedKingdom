# Особистий веб-сайт Валерії Усачової

## Опис проекту

Це особистий веб-сайт студентки факультету міжнародної торгівлі та права, група 1-7, який містить інформацію про студента, резюме та аналітичний звіт з економічної безпеки України.

## Структура проекту

Проект складається з наступних файлів:

- `index.html` - Головна сторінка з інформацією про студента
- `resume.html` - Сторінка з резюме
- `report.html` - Сторінка з аналітичним звітом
- `style.css` - Файл зі стилями для всіх сторінок
- `README.md` - Цей файл з описом проекту

## Як завантажити на GitHub

### Крок 1: Створення репозиторію

1. Увійдіть на [GitHub](https://github.com)
2. Натисніть на кнопку "New" (зелена кнопка) або "+" у правому верхньому куті
3. Виберіть "New repository"
4. Введіть назву репозиторію (наприклад, `student-website`)
5. Оберіть "Public" або "Private"
6. Натисніть "Create repository"

### Крок 2: Завантаження файлів

#### Варіант А: Через веб-інтерфейс GitHub

1. На сторінці вашого нового репозиторію натисніть "uploading an existing file"
2. Перетягніть всі файли (index.html, resume.html, report.html, style.css, README.md)
3. Додайте коментар до коміту (наприклад, "Initial commit")
4. Натисніть "Commit changes"

#### Варіант Б: Через Git командний рядок

```bash
# Ініціалізуйте Git у папці з файлами
git init

# Додайте всі файли
git add .

# Створіть коміт
git commit -m "Initial commit"

# Додайте віддалений репозиторій
git remote add origin https://github.com/ваш-username/student-website.git

# Завантажте файли
git push -u origin main
```

### Крок 3: Активація GitHub Pages

1. Перейдіть в налаштування репозиторію (Settings)
2. У лівому меню виберіть "Pages"
3. У розділі "Source" виберіть "main" branch
4. Натисніть "Save"
5. Через кілька хвилин ваш сайт буде доступний за адресою: `https://ваш-username.github.io/student-website/`
